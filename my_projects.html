My Projects

Previously I use to develop application with the block based coding softwares like MIT App Inventor and its famous distributions like Niotron, Appzard etc. When I started learning Java I got to know about extension development for these builders. Soon I started extension development and got one of my extension(TicTacToe Extension) with a good number of downloads. My extensions are as follows...

TicTacToe Extension

This extension simplifies the TicTacToe game for MIT App Inventor and its distributions.

QR Code Generator Extension

Dedicated to MIT App Inventor, this extension facilitates easy QR Code generation.

ColorSeekbar

This is also dedicated to the MIT App Inventor and its distributions for picking up the color.

TextEditor Extension

This extension was made to make the text editing process easier by adding different fonts etc.

DistanceFinder Extension

Calculate the distance between two places on Google Maps with this extension.

Pin Viewer Extension

Enhance your MIT App Inventor application's UI with a Pin or OTP View using this extension.
